# Sannith's Portfolio

A personal portfolio website for **P. Sannith**, a Backend Engineer and Data Analyst. This project showcases Sannith's skills, projects, and professional background.

## Features

- **Hero Section**: A visually engaging introduction.
- **About Me**: Detailed professional background and expertise.
- **Skills**: Highlighted technical skills including Backend, Databases, and Frontend technologies.
- **Projects**: Showcase of key projects with links to GitHub and demos.
- **Contact**: Contact information and social media links.

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Animations**: [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Icons**: [Font Awesome](https://fontawesome.com/), [Boxicons](https://boxicons.com/)
- **Styling**: Vanilla CSS with a focus on responsive design and modern aesthetics.

## Project Structure

```text
portfolio/
├── assets/             # Images, icons, and videos
│   ├── icon/           # SVG icons for skills
│   └── images/         # Project images and background video
├── index.html          # Main entry point
├── script.js           # Interactive elements and menu logic
├── style.css           # Custom styling
└── README.md           # Project documentation
```

## How to Run

1. Clone the repository or download the source code.
2. Open `index.html` in any modern web browser.
